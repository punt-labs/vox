# Usability

Measurable usability evaluation.

- Heuristic evaluation (Nielsen's 10): visibility, match, consistency, error prevention
- Cognitive walkthrough: per-step prediction of user action
- Usability testing: think-aloud, scenario-based, moderated and unmoderated
- Quantitative metrics: time-on-task, error rate, success rate
- Reporting findings as falsifiable claims, not opinions
