# Type Systems

Foundational reasoning about type theory in production languages.

- Soundness vs. ergonomics trade-offs (TypeScript, Swift, C#)
- Generics, conditional types, mapped types, type-level programming
- Subtyping vs. structural typing, variance, nominal vs. structural
- Type inference: when it helps, when it surprises, when to annotate
- Designing types that document intent, not just shape
